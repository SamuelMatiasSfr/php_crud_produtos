# Sistema de Gerenciamento de Produtos

Sistema em PHP para gerenciar produtos, permitindo cadastro, edição e exclusão de registros. Este projeto foi desenvolvido como atividade prática de Aplicações para WEB 2 no curso técnico de informática do CEFET-MG.

## Funcionalidades:
- Cadastrar produtos
- Editar produtos
- Excluir produtos
- Listar produtos em uma tabela 

## Tecnologias:
- PHP
- MySQL
- VS Code

## Como executar
1. Baixe e extraia o arquivo zip do projeto
2. Copie a pasta principal do projeto (pasta **crud_produtos**) para a pasta **C:\xampp\htdocs**
3. Abra o SGBD de sua preferência e rode o script do arquivo: **script_banco.txt**
4. Abra o navegador e digite no campo de pesquisa **localhost/crud_produtos/tela_produtos.php**
5. Teste o projeto
